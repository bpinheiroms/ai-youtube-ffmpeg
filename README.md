# ai-youtube-ffmpeg

Summary:
- Download video from youtube
- Convert to Mp3
- Transcribe
- Translate
- Resume
- Create PDF with the resume

How to run:
- pnpm start 'YOUTUBE_LINK' 'DESCRIPTION TO HELP THE IA' 'OPENAI_TOKEN'

e.g: pnpm start 'https://youtu.be/link' 'it's a link about X thing' 's-1234'
